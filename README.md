# AI File Organizer

An advanced Python automation tool that automatically organizes files into categorized folders based on file type.

---

# Features

✅ Automatically organizes files  
✅ Supports images, videos, audio, documents, archives, code files, applications, and fonts  
✅ Creates folders automatically  
✅ Handles unknown files using an "Others" folder  
✅ Fast and lightweight  
✅ Beginner-friendly Python automation project

# Installation

Clone the repository:

```bash
git clone https://github.com/YOUR-USERNAME/ai-file-organizer.git
```

Enter the project folder:

```bash
cd ai-file-organizer
```

Run the script:

```bash
python3 organizer.py
```

---

# How It Works

The script scans your Downloads folder and automatically moves files into categorized directories such as:

- Images
- Videos
- Documents
- Code
- Archives
- Audio
- Applications
- Others

---
